Inter Company Transaction

Inter-company functionality enables the automated flow of transactions between one company of an entity and another company of the same entity.

Inter Company Rules

In order to configure the inter company rules settings, Goto Settings -> General Settings -> Inter Company Rules.
SO and PO Transactions
Create Invoice/Refunds

Sale Order

When an SO is created for current company, the system will create a PO for other company which is associated to the current company.

Invoice

When a Customer Invoice is created for current company, the system will create a Vendor Bill for other company which is associated to the current company.
